# D.aise: An Automated README Generator from Source Code Repositories

D.aise is a platform designed to automatically generate README files from source code repositories using LLMs.

It includes a Python backend and a Next.js frontend.

The system is built to produce automated README documentation and supports both local LLM integration and remote LLM services such as Gemini, using an API key.

## Features

- Automatic README generation from source code repositories  
- Backend implemented in Python  
- Frontend built with Next.js  
- Integration with local LLMs  
- Support for remote LLMs such as Gemini via API key
