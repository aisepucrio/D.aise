## Configuração de Uso de LLM

Por padrão, este projeto está configurado para **não utilizar um LLM**.

Para habilitar o uso de LLM, é necessário ajustar a variável de ambiente `USE_LLM`.

### Como habilitar

Você pode fazer isso de uma das seguintes formas:

1. **Alterando o valor existente**  
   Adicione ou altere a variável USE_LLM na .env para receber true:
   ```env
   USE_LLM=TRUE
   ```
